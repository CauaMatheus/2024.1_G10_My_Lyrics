# 3.4. Participações - Padrões de Projeto

Essa entrega foi uma corrida contra o tempo. Além de entregarmos o modulo 3, tivemos que atualizar tudo das entregas 1 e 2 que não foram corrigidas anteriormente.

Para a entrega do módulo 3, primeiramente o grupo se dividiu para estudarmos individualmente os design patterns. Após esse tempo de estudos, resolvemos juntar ideias de Clean Architecture com os Design Patterns, já que essa será a arquitetura que provavelmente iremos seguir.

Em geral, todos perceberam que muitas coisas do Clean Architecture tinham como base os Design Patterns. Então acabamos usando como base para muitas coisas.

No fim, todos conseguiram participar em equipe de todos os focos. Porém, admitimos que a falta de poder dividir entre grupos foi um desafio.<br/>
Coisas que poderiam ser feitas de forma rápida demoraram por conta de ter que achar um tempo livre onde todos poderiam estar presentes.

Além da entrega do módulo 3, todos os módulos anteriores receberam o registro de alterações e suas referências. O protótipo foi melhorado e as páginas principais foram re-escritas para melhor entendimento.
